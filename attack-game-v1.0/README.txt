"Attack" game written by Christian Brookfield.

This version is compiled for the Mignonette Game Console (ATMEGA88).

Copyright 2008 Christian W. Brookfield
http://www.evilmadscientist.com/staticpages/index.php/contact

Instructions:

You will need the avr-gcc environment installed on your computer.

Rename Makefile.txt to Makefile, and type:

    make

If there are no errors, then connect the programmer to the Mignonette,
and type:

    make program

Enjoy!
